## ğŸ‡ºğŸ‡¸ English Version: Course Overview

1. **Cloud Concepts:** Why we use Cloud. High availability, global reach, and the agility to launch resources in seconds.
2. **Global Infrastructure:** The physical world of AWS. How **Regions** and **Availability Zones** keep your apps running even during a disaster.
3. **IAM & Security:** The "Identity" center. Managing users and the "Least Privilege" rule to keep the hackers out.
4. **EC2 (Compute):** Virtual servers. We use **User Data** scripts to automatically set up a web server at launch.
5. **S3 (Storage):** The infinite hard drive. Storing files (objects) and managing them with **Lifecycle Policies**.
6. **Databases:** Understanding structured data with **RDS** (SQL) versus the lightning-fast, scalable **DynamoDB** (NoSQL).
7. **Scaling & Traffic:** Using **Load Balancers** to distribute traffic and **Auto Scaling** to add servers during peak hours.
8. **Networking:** Building your own private cloud with **VPC**, managing domains with **Route 53**, and connecting offices via **Transit Gateway**.
9. **Serverless:** The modern way. Running code with **AWS Lambda** and building APIs with **API Gateway** without managing any servers.
10. **Monitoring & Observability:** Performance tracking with **CloudWatch**, activity logging with **CloudTrail**, and debugging with **X-Ray**.
11. **Migration:** Moving terabytes of data physically with the **Snow Family** or digitally with **DMS** (Database Migration Service).
12. **Advanced Security:** Protecting websites with **WAF** (Firewall), stopping DDoS with **Shield**, and encrypting data with **KMS**.
13. **Cost Management:** Forecasting bills with **Cost Explorer**, setting limits with **Budgets**, and getting expert tips from **Trusted Advisor**.
14. **Infrastructure as Code (IaC):** Writing your cloud like code using **Terraform** or **CloudFormation**. One click to deploy everything.
15. **DevOps / CI/CD:** Automation at its best. Moving code from developer to production automatically with **CodePipeline** and **CodeDeploy**.
